# SmartAccountant - Database Setup Guide
# مدقق الحسابات الذكي - دليل إعداد قاعدة البيانات

## الملفات الأساسية (Core Files)

### 1. `00_Create_Database_Schema.sql` - إنشاء قاعدة البيانات والجداول
- **الغرض**: إنشاء قاعدة البيانات الكاملة مع جميع الجداول والعلاقات
- **المحتويات**:
  - إنشاء قاعدة البيانات بجودة `Arabic_100_CI_AS`
  - 15 جدول أساسي:
    - Currencies (العملات)
    - Plans (الخطط)
    - Roles (الأدوار)
    - Permissions (الصلاحيات)
    - Accounts (الحسابات)
    - Users (المستخدمون)
    - Countries (الدول)
    - Customers (العملاء)
    - Invoices (الفواتير)
    - PhoneNumbers (أرقام الهاتف)
    - SystemSettings (الإعدادات)
    - ActivityLogs (سجلات النشاط)
    - وغيرها...

**التعليمات**:
```sql
-- في SQL Server Management Studio
USE [master];
-- قم بتشغيل الملف كاملاً
```

---

### 2. `00_Cleanup_Corrupted_Data.sql` - تنظيف البيانات الفاسدة
- **الغرض**: حذف أي بيانات فاسدة من الجداول السابقة
- **المحتويات**:
  - حذف جميع البيانات من جميع الجداول
  - إعادة تعيين عدادات الهوية (IDENTITY)
  - إعادة تشغيل النظام

**التعليمات**:
```sql
-- قبل إضافة البيانات الجديدة
USE [SmartAccountant_v1005_DB];
-- قم بتشغيل هذا الملف أولاً
```

---

### 3. `01_Insert_Base_Data.sql` - إدراج البيانات الأساسية الكاملة
- **الغرض**: إضافة جميع البيانات الأساسية اللازمة لبدء التطبيق
- **المحتويات**:

#### أ) العملات (12 عملة)
- العملات العربية: SAR, AED, KWD, QAR, OMR, BHD, JOD, LBP, EGP
- العملات الدولية: USD, EUR, GBP
- الافتراضي: SAR (الريال السعودي)

#### ب) الخطط (3 خطط)
```
1. الخطة الأساسية - 99 ريال/شهر
   - حساب واحد، 3 مستخدمين، 100 فاتورة، 50 عميل

2. الخطة المحترفة - 299 ريال/شهر
   - 3 حسابات، 10 مستخدمين، 1000 فاتورة، 500 عميل

3. خطة المؤسسات - 999 ريال/شهر
   - 10 حسابات، 50 مستخدم، 10000 فاتورة، 5000 عميل
```

#### ج) الأدوار (6 أدوار)
- مسؤول النظام - صلاحيات كاملة
- مسؤول الحساب - إدارة الحساب الكامل
- مدير العمليات - إدارة الفواتير والعملاء
- موظف المبيعات - إدارة العملاء
- موظف الحسابات - إدارة الفواتير والتقارير
- عارض البيانات - عرض البيانات فقط

#### د) الصلاحيات (18 صلاحية)
مقسمة حسب الفئات:
- **المستخدمين**: عرض، إضافة، تعديل، حذف
- **العملاء**: عرض، إضافة، تعديل، حذف
- **الفواتير**: عرض، إنشاء، تعديل، حذف، طباعة
- **الإعدادات**: عرض، تعديل
- **التقارير**: عرض، تصدير

#### هـ) الدول (20 دولة عربية)
- بلاد الخليج: SA, AE, KW, QA, OM, BH
- بلاد الشام: JO, LB, SY, IQ, PS
- شمال أفريقيا: EG, LY, TN, DZ, MA, MR
- شرق أفريقيا: SD, DJ, SO

#### و) حساب الإدارة
- اسم: حساب الإدارة
- البريد: admin@smartaccountant.local
- الهاتف: +966123456789
- العملة الافتراضية: SAR
- الخطة: خطة المؤسسات (الكاملة)

#### ز) مستخدم مسؤول
- اسم المستخدم: **admin**
- كلمة المرور: **admin123**
- الدور: مسؤول النظام
- اللغة: العربية

#### ح) الإعدادات النظامية (12 إعداد)
- اسم التطبيق: مدقق الحسابات الذكي

## ملاحظات مهمة

### ✓ دعم اللغة العربية
- جميع النصوص العربية محمية بحرف `N` قبلها
- قاعدة البيانات تستخدم جودة `Arabic_100_CI_AS`
- جميع الحقول التي تحتوي نصوص عربية هي `NVARCHAR`

### ✓ الأمان
- كلمة مرور المسؤول مشفرة بـ bcrypt
- لا تستخدم المسؤول مباشرة في الإنتاج
- غيّر كلمة المرور بعد الإعداد الأول

### ✓ قابلية التوسع
- يمكن إضافة المزيد من الدول والعملات
- يمكن تخصيص الأدوار والصلاحيات حسب احتياجاتك
- يمكن إضافة حسابات وموظفين جدد بسهولة

### ✓ الملفات المؤرشفة
- جميع الملفات القديمة موجودة في مجلد `archived/`
- احتفظ بها للرجوع إليها عند الحاجة

## التحقق من الإعداد

### التحقق من إنشاء الجداول
```sql
USE SmartAccountant_v1005_DB;
SELECT TABLE_NAME FROM INFORMATION_SCHEMA.TABLES 
WHERE TABLE_TYPE = 'BASE TABLE' 
ORDER BY TABLE_NAME;
```

### التحقق من البيانات
```sql
-- عدد العملات
SELECT COUNT(*) AS [عدد العملات] FROM Currencies;

-- عدد الخطط
SELECT COUNT(*) AS [عدد الخطط] FROM Plans;

-- عدد الأدوار
SELECT COUNT(*) AS [عدد الأدوار] FROM Roles;

-- البيانات العربية
SELECT TOP 3 Code, Name FROM Countries WHERE IsActive = 1;

-- حساب المسؤول
SELECT * FROM Accounts WHERE Name = N'حساب الإدارة';

-- مستخدم المسؤول
SELECT Username, FullName, IsActive FROM Users WHERE Username = N'admin';
```

## المشاكل الشائعة والحلول

### المشكلة: رسالة خطأ "Collation not supported"
**الحل**: تأكد من استخدام نسخة SQL Server أحدث تدعم `Arabic_100_CI_AS`

### المشكلة: النصوص العربية تظهر بشكل مشوه
**الحل**: تأكد من تشغيل جميع الملفات بنجاح وأن جودة قاعدة البيانات هي `Arabic_100_CI_AS`

### المشكلة: تعذر الاتصال بـ (localdb)\mssqllocaldb
**الحل**: 
1. تأكد من تثبيت SQL Server LocalDB
2. نفّذ: `sqllocaldb start mssqllocaldb`
3. تحقق من الاتصال: `sqlcmd -S "(localdb)\mssqllocaldb" -Q "SELECT @@VERSION"`
