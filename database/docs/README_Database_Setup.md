## التطبيق على الإنتاج

### خطوات النشر على السيرفر

1. **التحضير:**
   ```
   - تأكد من وجود SQL Server
   - اختبر الاتصال بـ SQL Server
   - احفظ نسخة احتياطية من أي بيانات قديمة
   ```

2. **التثبيت:**
   ```
   أ. شغل: 01_Database_Schema.sql
   ب. شغل: 02_Base_Data.sql
   ```

3. **التحقق:**
   ```sql
   -- تحقق من إنشاء الجداول
   SELECT TABLE_NAME FROM INFORMATION_SCHEMA.TABLES 
   WHERE TABLE_SCHEMA = 'dbo'
   ORDER BY TABLE_NAME;
   
   -- تحقق من البيانات الأساسية
   SELECT COUNT(*) FROM Currencies;      -- يجب أن يكون 20
   SELECT COUNT(*) FROM Permissions;     -- يجب أن يكون 52
   SELECT COUNT(*) FROM SystemSettings;  -- يجب أن يكون 7+
   ```

4. **الاتصال:**
   - حدث connection string في التطبيق
   - اختبر الاتصال
   - ابدأ التطبيق
