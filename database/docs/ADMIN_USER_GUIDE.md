## 🔑 كلمات مرور آمنة:

### الكلمة الحالية:
- **Username:** admin
- **Password:** admin123

⚠️ **تنبيه أمني مهم:**
1. **غيّر كلمة المرور فوراً** بعد الدخول الأول
2. **لا تستخدم كلمات مرور ضعيفة** في الإنتاج
3. **استخدم كلمات مرور قوية** تحتوي على:
   - أحرف كبيرة
   - أحرف صغيرة
   - أرقام
   - رموز خاصة (!@#$%^&*)
